# Fotos del Equipo Editorial

## Ubicación Correcta
Las fotos deben estar en: `public/equipo/`

## Archivos Necesarios

1. ✅ `Omar.png` - Dr. Omar Armando Villota (ya existe)
2. ⏳ `maria-gonzalez.jpg` - Dra. María González López
3. ⏳ `roberto-silva.jpg` - Dr. Roberto Silva Martínez
4. ⏳ `ana-rodriguez.jpg` - Lic. Ana Rodríguez Gómez

## Importante
- En Angular 17+, los archivos en la carpeta `public` se sirven directamente
- No uses el prefijo `assets/` en las rutas
- Las rutas en el código son: `equipo/nombre-archivo.jpg`
- Si agregas nuevas fotos, colócalas en `public/equipo/`

## Especificaciones de las Fotos
- Formato: JPG o PNG
- Tamaño mínimo: 300px × 300px
- Aspecto: Cuadrado (1:1)
- Calidad: Alta resolución
